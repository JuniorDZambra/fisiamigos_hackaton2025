Video explicativo del proyecto
Juntar en un directorio completo los archivos requirements.txt,Provincias.geojson,NDVI_Estadisticas_Provincias.csv y app.py. Luego, crear un subdirectorio, llamado assets e inclui dentro styles.css
Una vez hecho lo anterior, si se tiene instalado python en el ordenador o anaconda, abrir el prompt (CMD O Windows powershell) y buscar la ruta de la carpeta principal, en el caso de eeste proyecto, fue cd C:\Users\d\Documents\fisiamigos-reto-3.
Ejecutar pip install -r requirements.txt
luego python app.py
